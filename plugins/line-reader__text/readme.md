# Line Reader

Reads a line off of a text file. That's it.

![illustration](image.png)


## Example Usage
```
python line-reader__text.py \
  --text-file-input="media/audio" \
  --c
```
## Arguments

`--text-file-input`

The source text file for the script.

`-c`

Adding this flag prints the output in a console.

# blend__audio
Blends a variable number of audio tracks

![illustration](image.jpg)

## Example Usage

```
# Blend Audio
# //////////////////////////////

print "blending audio..."

python random-blend__audio.py \
  --audio-input-dir="media/audio" \
  --number-of-files="media/audio" \
  --clip-duration="(30,60)" \
  --output="path/to/output" \
 &&
 ```


`--audio-input-dir="media/audio"`
`--number-of-files="5"`
`--clip-duration="(30,60)"`
`--output="path/to/output"`

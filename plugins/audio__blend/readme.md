# random-blend__audio
Blends a randomly selected, variable number of audio tracks

## Example Usage

```
# Blend Audio
# //////////////////////////////

print "blending audio..."

python random-blend__audio.py \
  --audio-input-dir="media/audio" \
  --number-of-tracks="media/audio" \
  --clip-duration="(30,60)" \
  --output="path/to/output" \
 &&
 ```
